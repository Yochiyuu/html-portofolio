Tentang Saya

Saya Andrew, mahasiswa aktif di Universitas Tarumanagara yang sedang mendalami dunia Web Development. Ketertarikan saya terhadap teknologi web berawal dari rasa ingin tahu tentang bagaimana sebuah situs dapat bekerja secara dinamis dan responsif. Sejak itu, saya mulai membangun berbagai proyek berbasis web yang menggabungkan desain fungsional, performa optimal, dan pengalaman pengguna yang menyenangkan.


Saya fokus pada pengembangan antarmuka yang intuitif, integrasi backend yang efisien, serta penerapan praktik terbaik dalam pengembangan web modern. Dalam proses belajar, saya aktif menggunakan HTML, CSS, dan JavaScript, serta mulai mendalami framework seperti React dan Vue. Di sisi backend, saya mempelajari Node.js dan Express, serta mengelola database menggunakan MySQL dan MongoDB.


Setiap proyek yang saya kerjakan menjadi sarana untuk mengasah kemampuan teknis dan kreativitas saya. Tujuan saya adalah menjadi web developer yang tidak hanya mahir secara teknis, tetapi juga mampu memahami kebutuhan pengguna dan menciptakan solusi digital yang berdampak.
